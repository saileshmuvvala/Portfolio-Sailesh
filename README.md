# Sailesh Portfolio (React)

This is a personal portfolio website built using **React.js and CSS**. The project showcases my profile, technical skills, projects, work experience, and contact information in a modern and responsive interface.

The application is structured using reusable React components and organized data files, making the project modular and easy to maintain.

## Getting Started

### Installation

Clone the repository

```
git clone https://github.com/saileshmuvvala/Portfolio-Sailesh.git
```

Navigate into the project folder

```
cd Portfolio-Sailesh
```

Install dependencies

```
npm install
```

Start the development server

```
npm start
```

The app will run at:

```
http://localhost:3000
```
